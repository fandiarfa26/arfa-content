# DESIGN.md

# Design Principles

* Minimal.
* Cepat.
* Fokus pada konten.
* Sedikit klik.
* Mobile friendly.

---

# Navigation

* Dashboard
* Backlog
* Weekly Plan
* Recap

---

# Dashboard

Menampilkan:

* Total Ideas
* Total Ready
* Total Planned
* Total Published
* Category Statistics
* Current Weekly Plan

---

# Backlog

Tampilan:

Idea | Ready | Planned | Published

Kanban hanya visualisasi.

Status tidak dapat diubah manual.

Card berpindah otomatis berdasarkan data.

---

# Card Design

Card hanya menampilkan:

* Judul

Category ditampilkan sebagai border kiri.

* Hijau → npm
* Biru → run
* Kuning → test
* Abu-abu → lainnya

---

# Content Editor

Single long page.

Urutan:

1. Title
2. Category
3. Hook
4. Problem
5. Experience
6. Lesson
7. CTA
8. Caption
9. Hashtags
10. Publish Information

# Weekly Plan

Memilih konten dari backlog.
